Jason Statham API

Available Endpoints

  • /info.json - Basic information about Jason Statham
  • /movies.json - List of movies starring Jason Statham
  • /awards.json - Awards and nominations

How to Use

Access any endpoint directly via URL:

https://vaernewijck.github.io/json-statham/api/movies.json

Example: Fetch movies with JavaScript

const fetchData = async () => {
  try {
    const response = await fetch('https://vaernewijck.github.io/json-statham/api/movies.json');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Error:', error);
  }
};

fetchData();
